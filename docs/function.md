# 功能使用

Leet-Robot拥有多个指令与功能

## 基础指令(私聊/群聊) {#基础指令}
- 指令

    展示目前robot所拥有的功能指令

- 关于

    展示robot的各项信息

## 搜图功能(私聊/群聊) {#搜图功能}
采用三大搜图引擎`(sacueNao/ascii2d/iqdb)`
- 使用

1. 发送`搜图`给robot(私聊) or 发送`搜图`到群里(群聊)

2. robot回复`请发送一张图片`后

3. 发送一张图片

4. 等待robot返回搜图结果

## steam特别优惠(群聊) {#steam特别优惠}

- 使用

发送`steam特别优惠`到群里

等待robot返回数据

## AI回复(群聊) {#AI回复}

- 使用

@robot并附带问题文字

等待robot回复

## 正常图(私聊) {#正常图}

- 使用

发送`来点图`给robot

等待robot发送图片

## se图(私聊) {#se图}

- 使用

发送`来点涩图`给robot

等待robot发送图片
